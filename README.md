# OpenCart Manual Testing Project

## Project Overview

This project contains manual testing documentation and defect tracking for the OpenCart e-commerce application.

## Modules Tested

* Login
* Registration
* Search
* Wishlist
* Shopping Cart
* Checkout
* Order History
* Hero Banner/Slider

## Testing Types Performed

* Functional Testing
* UI Testing
* Negative Testing
* Validation Testing

## Tools Used

* Jira
* Excel
* GitHub

## Artifacts Included

* Test Scenarios
* Test Cases
* Defect Reports
* Jira Bug Screenshots
